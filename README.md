# City Temperatures

A simple web app to compare monthly temperature data for cities around the world.

## Features

- Compare up to 5 cities on an interactive line chart
- Toggle between Celsius and Fahrenheit
- Export chart as high-quality PNG
- Mobile-friendly responsive design

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Recharts
- shadcn/ui

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
